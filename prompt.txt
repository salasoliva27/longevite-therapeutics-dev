  - ScrollTrigger: https://cdnjs.cloudflare.com/ajax/libs/gsap/3.12.5/ScrollTrigger.min.js

- Intersection Observer API as fallback/enhancement
- Font: Google Fonts - **Inter** or **Outfit** for headings and body

### Animation Philosophy (Medical-Appropriate)
**"Sophisticated Minimalism"** - Animations should be:
- **Subtle and slow** - Never jarring or distracting
- **Professional** - Build trust, not entertainment
- **Purposeful** - Guide attention to important information
- **Accessible** - Respect `prefers-reduced-motion`
- **Smooth** - Premium feel, like Apple or high-end medical sites

### Specific Animation Rules:
1. **Parallax scrolling** - VERY subtle (max 20% speed difference), only on decorative elements
2. **Fade-in on scroll** - Gentle opacity 0 → 1 over 0.8s
3. **Slide-up animations** - Small movement (20-30px max) with ease-out
4. **Stagger animations** - 100-150ms delays between elements
5. **Hover effects** - Minimal (2-4px lift, subtle shadow/glow)
6. **NO spinning, bouncing, or aggressive motion**
7. **Scroll progress indicator** - Thin, subtle line at top

### JavaScript Animation Pattern
```javascript
// Intersection Observer for scroll animations
const observerOptions = {
  threshold: 0.2,
  rootMargin: '0px 0px -50px 0px'
};

const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      entry.target.classList.add('animate-in');
    }
  });
}, observerOptions);

// Parallax effect implementation
window.addEventListener('scroll', () => {
  const scrolled = window.pageYOffset;
  const parallaxElements = document.querySelectorAll('.parallax');
  parallaxElements.forEach(element => {
    const speed = element.dataset.speed || 0.5;
    element.style.transform = `translateY(${scrolled * speed}px)`;
  });
});
```
